## ✨ Features

- **🗂️ 95+ Categories** — Templates organized by industry and use case
- **🔍 Live Search & Filter** — Find templates instantly by name or category
- **📱 Fully Responsive** — Works beautifully on desktop, tablet, and mobile
- **🎨 3D Card Effects** — Interactive tilt and glow effects powered by VanillaTilt
- **⚡ Lazy Loading** — Cards load progressively with scroll-based reveal animations
- **🔄 7 Curated Collections** — Sourced from top template repositories across GitHub
- **🌓 Modern Glassmorphism UI** — Dark theme with animated gradient blobs

## 🛠️ Development

### Prerequisites

- Node.js (for running `update_projects.js`)

### Regenerate Project Data

```bash
node update_projects.js
```

This scans all template directories and regenerates `projects_data.js`.

### Customization

- **Colors & Theme** — Edit CSS variables in `style.css` (`:root` block)
- **Cards per page** — Change `ITEMS_PER_PAGE` in `script.js`
- **Header content** — Edit text in `index.html`
